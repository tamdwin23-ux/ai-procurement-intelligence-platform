# 🎫 AI Customer Support Ticket Classifier

> An AI-powered customer support automation system that classifies incoming tickets, detects urgency and sentiment, retrieves relevant knowledge, routes requests to the correct team, and sends automated responses using **n8n, OpenAI, Supabase, PostgreSQL, and RAG**.

---

## 🚀 Business Problem

Customer support teams receive large volumes of requests through email, website forms, chat, and other channels.

Manually reviewing and routing every ticket can lead to:

- Slow response times
- Incorrect ticket assignment
- Missed urgent cases
- Inconsistent customer communication
- Repetitive support work
- Poor visibility into common customer issues

---

## 💡 Solution

This system uses AI to analyse each incoming support request and turn it into structured, actionable data.

The workflow:

- Captures incoming support tickets
- Identifies the customer and issue
- Classifies the ticket category
- Detects customer sentiment
- Measures urgency and priority
- Searches approved company knowledge using RAG
- Routes the ticket to the correct department
- Creates a structured support record
- Notifies the support team
- Sends an acknowledgement to the customer
- Escalates sensitive or high-priority cases to a human
